The Attendance Management System is a user-friendly web application developed using Python and Streamlit to simplify the process of managing student attendance. The system provides an efficient way to maintain student records, record daily attendance, and monitor attendance statistics through an interactive graphical interface.

The application consists of several modules, including student management, attendance recording, attendance viewing, and attendance percentage calculation. Users can easily add new students to the system, view the list of registered students, mark students as present or absent on a specific date, and access attendance records whenever required. All data is stored in text files, making the application lightweight and easy to deploy without requiring a database.

One of the key features of the system is the automatic calculation of attendance percentages. By analyzing attendance records, the application determines the total number of classes attended by a student and calculates their attendance percentage. This helps teachers and administrators quickly evaluate student participation and attendance performance.

The project is designed with a simple and intuitive interface using Streamlit widgets such as text inputs, radio buttons, date selectors, and sidebar navigation menus. This ensures that users can interact with the system without any technical expertise. The application also includes validation messages and notifications to improve the user experience.

This project demonstrates the practical implementation of Python concepts such as file handling, conditional statements, loops, and data processing, along with web application development using Streamlit. It serves as an excellent beginner-to-intermediate level project for students learning Python and web development. The Attendance Management System can be further enhanced by integrating a database, authentication system, attendance reports, search functionality, and data visualization features to create a more robust and scalable solution for educational institutions.
